# Country Tech Image Optimizer

A professional-grade batch image processing toolkit with:

- CLI image resizing
- Desktop GUI application
- Format conversion (jpg, png, webp)
- Compression optimization
- Aspect ratio control
- Batch folder processing
- Progress tracking

## Installation

pip install -r requirements.txt

## CLI Usage

python image_resizer_cli.py input_folder output_folder --width 800 --height 600

## GUI Usage

python image_resizer_gui.py
